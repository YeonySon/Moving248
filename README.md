# 이사할 땐 248 - 비대면 이사 견적 서비스

## 프로젝트 진행 기간
SSAFY 9기 2학기 공통 프로젝트
2023.07.03(월) ~ 2023.08.18(금) 7주간 진행
<br/><br/>
## 프로젝트 소개
- 프로젝트 명 : 비대면 이사 견적 서비스
- 주요 기능
    - 영상등록을 통한 이사 신청서 간소화
    - 업체들의 견적 등록을 통한 가격 비교
    - 실시간 화상 상담
- 주요 기술
    - React.js
    - Spring boot
    - WebRTC
    - AWS S3
    - EC2
    - Redis
<br/><br/>
